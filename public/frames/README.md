# Moldura de Líder

Coloque aqui o arquivo `lider.png` (fundo transparente) com a moldura
que aparece ao redor da foto de quem está em 1º lugar no ranking de pontos.

Ela é exibida no menu lateral, sobreposta à foto de perfil (que tem 160x160px).
Use uma imagem com a parte circular centralizada e proporção aproximadamente
quadrada — os elementos decorativos (coroa, folhas, etc.) podem ultrapassar
o círculo central, eles vão aparecer por cima/ao redor da foto.

Se o arquivo não existir, o sistema exibe um emoji de coroa 👑 no lugar.

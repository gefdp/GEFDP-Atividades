# Logo GEFDP

Coloque aqui o arquivo `logo.png` com a logo do GEFDP (fundo transparente,
de preferência na cor roxa usada na marca).

Ela é usada:
- No menu lateral, ao lado de "Atividades"
- Nas telas de login/cadastro, acima do título

Se o arquivo não existir, o sistema exibe um ícone genérico no lugar.

# Medalhas das ferramentas

Coloque aqui os PNGs das medalhas/insígnias de cada ferramenta. O nome do
arquivo precisa ser exatamente o `id` da ferramenta (veja
`src/lib/toolsCatalog.js`), dentro da pasta da categoria correspondente.

Se o arquivo não existir, a medalha exibe um ícone genérico no lugar — então
pode ir adicionando aos poucos.
